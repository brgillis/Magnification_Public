39  /************************************************************
40  interpolator_derivative
41  -----------------
42 
43  This class operates like an interpolator with added
44  features. The same functions can be used as with the basic
45  interpolator class, but this class also has the ability to point
46  to a different interpolator, which this is intended to be
47  the derivative of. Then, "unknown" domain points can be
48  passed to this class, and it will calculate the derivative of
49  the other interpolator at those points to help fill in gaps.
50 
51  The points passed to this can all be "known" (domain and range
52  passed), all "unknown" (only domain passed, range calculated),
53  or a mix of the two.
54 
55  Unknown points are calculated using a smoothing kernel to help
56  handle noise in the pointed-to interpolator. This must be adjusted
57  by hand based on how noisy the interpolator's points are for optimal
58  results. The noisier it is, the wider the kernel should be.
59  Use the set_sample_scale(...) and set_sample_max_width(...)
60  functions to adjust the kernel size (the scale is the sigma of
61  a Gaussian, the max_width is the limits of integration). Both
62  take in values as representing a fraction of t_max - t_min.
63 
64  \************************************************************/
